\# Team Contribution Statement



\## Project Title

Prime Digitizers Service Management Portal



\## Team Members and Responsibilities



\### Ahsan Hamid - Backend Developer

Ahsan was responsible for designing and implementing the backend API using Node.js and Express.js. His work included backend project structure, REST API endpoints, authentication, JWT token handling, role-based access control, input validation, error handling, and MySQL database integration.



\### Erum - Frontend Developer

Erum was responsible for developing the React.js frontend interface. Her work included UI layout, reusable components, client-side routing, form handling, dashboard pages, responsive design, and frontend API integration using Axios.



\### Sujata - Database, Testing, and Documentation Lead

Sujata was responsible for designing the MySQL database schema, creating SQL scripts, preparing seed data, documenting test cases, collecting testing evidence, preparing the case study report, presentation slides, and contribution documentation.



\## Equal Contribution Declaration

All team members contributed to project planning, integration, testing, debugging, review, and final demonstration. The workload was divided based on frontend, backend, and database/documentation responsibilities.



\## Collaboration Method

The team used GitHub repositories under Ahsan Hamid’s GitHub account. Team members were added as collaborators and contributed through their assigned repositories.
